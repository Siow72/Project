# Generative Smart Resume Builder

A Python app that helps build resumes and recommend jobs using resume information.

## Files
- `resume_builder.py` - resume builder logic
- `job_recommendation.py` - job recommendation logic
- `pdf_generator.py` - PDF design logic
- `recommender.py` - recommendation helper
- `ui_components.py` - UI components
- `prompt.txt` - prompt text for LLM model

## How to run
```bash
pip install -r requirement.txt
python resume_builder.py
